# we.play

## Installation

### Windows

```
git clone https://github.com/pkociepka/we.play_algo
cd we.play_algo
set FLASK_APP=/api/endpoint.py
python -m flask run
```

### Linux

```
git clone https://github.com/pkociepka/we.play_algo
cd we.play_algo
export FLASK_APP=/api/endpoint.py
python -m flask run
```
